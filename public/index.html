    Post to TikTok
    
          
    
            🧪 Mode Sandbox (app TikTok en review)
            
    
              Tant que l'app est en review, TikTok n'autorise les posts que vers des tester accounts ajoutés dans la console Developer (Manage app → Sandbox → Add tester).
              Les vidéos uploadées sont automatiquement privées (SELF_ONLY). Une fois l'audit Content Posting API passé, tu pourras basculer en Direct + Public.
            
    
          
    
          
    
            Génère d'abord une vidéo UGC dans Video UGC — le bouton Post to TikTok y apparaît directement.
            Sinon viens ici pour poster manuellement n'importe quel URI Veo.
          
    
          
    
            Inbox : la vidéo arrive dans l'Inbox de l'app TikTok mobile, tu finalises caption/hashtags/musique puis publies à la main. Pas d'audit requis.
    
            Direct : poste live immédiatement avec la caption fournie. Requiert l'audit Content Posting API + scope video.publish.
          
    
          
    
            📍 Où trouver la vidéo si elle finit en Inbox ? (elle n'apparaît PAS dans Drafts du profile, c'est le piège classique)
            
      
              
    1. Force-quit puis rouvre l'app TikTok sur ton téléphone (clear le cache local, sinon la notif n'apparaît pas tout de suite)
      2. 
              
    2. Tap l'onglet Inbox en bas (icône enveloppe/cloche, juste à gauche du Profile)
      3. 
              
    3. Section Activity ou System Notifications en haut
      4. 
              
    4. Cherche une notif type :
                
        
                  
      • "You have a draft from an external app"
        • 
                  
      • "Continue editing your video"
        • 
                  
      • "sideloot uploaded a video"
        • 
                
      
              
      5. 
              
    5. Tap dessus → ouvre l'éditeur TikTok avec ta vidéo → caption/sound/cover → Post
      6. 
            
    
            
    Fallback si rien après ces étapes : ouvre TikTok Studio (Profile → ☰ en haut à droite → Studio → onglet Inbox). En sandbox certains tester accounts ne reçoivent pas les push notifs — c'est une limitation connue.
